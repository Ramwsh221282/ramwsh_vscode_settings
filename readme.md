### Installation guide:

1. Copy paste settings from `settings.json` to your local `settings.json`.
2. Place `custom` folder in your `.code` or `.code-insiders` folder.
3. The extensions installation should be followed by terminal command: `cat extensions.txt | xargs -L 1 code --install-extension` or if you use insiders version use: `cat extensions.txt | xargs -L 1 code-insiders --install-extension`
4. Copy paste settings from `keybindings.json` to your local `keybindings.json` settings
